Bewerbungstracker (Python)

Ein Konsolenprogramm zum Verwalten von Bewerbungen.

Funktionen:
- Offene Stellen anzeigen
- Bewerbung hinzufügen
- Bewerbungen in JSON speichern
- Bewerbungen anzeigen
- Bewerbungen löschen
- Bewerbungsdatum automatisch speichern

Technologien:
- Python
- JSON
- datetime
